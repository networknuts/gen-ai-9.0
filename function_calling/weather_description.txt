This function provides weather of a given location by providing 
the function with the zipcode of the given location.
The function requires zipcode as an argument, however user's often 
do not provide zipcodes rather they provide location names. 
If a location is received, we should provide the zipcode of that location
for this function to execute properly.

For example:
Input: what is the weather in new delhi?
Output: new delhi zipcode is 110001, so function should use 110001 as zipcode argument.
